- Go Language

## How to run API
1. Clone this project
2. Install PostgreSQL with Docker:
```bash
docker compose up -d
```
3. Run Program:
```bash
go run main.go 
```
4. Open SwaggerUI in browser by using URL:
```
http://localhost:8080/swagger/index.html
```
